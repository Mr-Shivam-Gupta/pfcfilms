# PFC Films Admin Panel

Admin panel for managing PFC Films website content.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file (optional):
```
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

The admin panel will be available at `http://localhost:3001`

## Features

- Dashboard with statistics
- Manage Productions
- Manage Courses
- Manage Gallery (Photos & Videos)
- View and manage Contact submissions
- Manage Celebrities
- Manage Testimonials
- Manage Awards
- Manage Stats
- Manage Services
- Manage About content

## Login

After setting up the backend, create an admin account using the register endpoint, then login with those credentials.
